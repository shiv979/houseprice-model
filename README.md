# House Price Prediction Models

This project explores different machine learning models for predicting house prices using the California Housing Dataset.

The main goal was to compare the performance of multiple regression models and identify the one that produced the most accurate predictions.

## Models Used

* Linear Regression
* Ridge Regression
* Decision Tree Regressor

## Evaluation Metrics

The models were evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Results

Among the evaluated models, the Decision Tree Regressor achieved the best overall performance and was selected as the final model.
